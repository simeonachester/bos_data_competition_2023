The Following are the Data Sets Provided for the Competition. These are Unweighted Subsets of the 2018/2019 MICS Data set for Guyana.

DataSets:
1.wm_Women 15 to 49_Stat_Day - Data Set for Surveyed Women in Reproductive age range 15 - 17
2.mn_Males 15 to 49_Stat_Day - Data Set for Surveyed Men in Reproductive age Range 15 - 17
3.tn_MosquitoNets_Stat_Day - Data Set for Surveyed Households who were checked for ownership and usage of Mosquito Nets
4.hl_Household_List_Stat_Day - Data Set for Surveyed Household; List of Household Members
5.hh_Household_Stat_Day - Data Set for Surveyed Households
6.fs_ChildOver 5 and below 18_Stat_Day - Data Set for Surveyed Households Primary Caretakers and/or Mothers of Children Between the Age 5 - 17
7.ch_ChildUnder 5_Stat_Day - Data Set for Surveyed Households Primary Caretakers and/or Mothers of Children below the Age 5
8.bh_BirthHistory_Stat_Day - Data Set for Surveyed Households' Women with a Birth History
